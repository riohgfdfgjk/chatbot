# 👟 SneakerBot v2 — Telegram-магазин кроссовок

Телеграм-бот на Python (aiogram 3) с красивым интерфейсом, каталогом с фото, корзиной-чеком, избранным, историей заказов и AI-ассистентом.

---

## 🗂 Структура проекта

```
sneaker_bot/
├── main.py          # Точка входа: Bot, Dispatcher, polling
├── handler.py       # Вся логика: команды, кнопки, обработчики
├── keyboard.py      # Клавиатуры (Reply + Inline, динамические)
├── data.py          # Общие данные: каталог, корзины, избранное, история
├── requirements.txt # Зависимости
└── README.md        # Документация
```

---

## ⚙️ Установка и запуск

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Указать токен (в main.py строка BOT_TOKEN или через env)
export BOT_TOKEN="ваш_токен"

# 3. Запустить
python main.py
```

## 🎨 Интерфейс и функционал

| Раздел | Описание |
|---|---|
| `/start` | Приветствие с фото и меню |
| `/help` | Полная справка |
| `/add Бренд Цена` | Добавить товар в каталог |
| `/search запрос` | Поиск по каталогу |
| **👟 Каталог** | Товары с эмодзи + кнопки ❤️ / 🛒 |
| **🛒 Корзина** | Прогресс-бар + красивый чек-таблица |
| **✅ Оформить заказ** | Подтверждение + сохранение в историю |
| **❤️ Избранное** | Сохранённые товары |
| **📋 История заказов** | Последние 10 заказов с датой |
| **📞 Контакты** | Телефоны и часы работы |
| **🤖 AI** | Ссылка на @icecream2508_bot |

### UX-улучшения
- Эффект «печатает...» (`send_chat_action`) перед каждым ответом
- Прогресс-бар корзины: `▓▓▓░░`
- Чек-таблица с псевдографикой (`┌─┬─┐`)
- Разделители `━━━━` для визуальной структуры
- Фото товара при выборе из каталога
- Кнопка ❌ Отмена при вводе количества
- Inline-кнопки на странице каждого товара
- Редактирование сообщений вместо новых (очистка корзины)

## 🏗 Архитектура данных

```python
catalog_items   # list[dict]  — товары: name, price, emoji, desc, photo
user_carts      # dict        — user_id → {item: {count, price}}
user_states     # dict        — user_id → {waiting_for_qty: name}
user_favorites  # dict        — user_id → set(names)
order_history   # dict        — user_id → list[str]
```

---

## 🛡 Обработка ошибок

- Некорректное количество → сообщение + кнопка Отмена
- Дубликат в `/add` → предупреждение
- Товар не найден → алерт / очистка состояния
- Фото недоступно → fallback на текст
- Пустая корзина при оформлении → алерт
